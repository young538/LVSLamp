# GUI 레이아웃 재배치 계획

## 1. 원칙

| 위치 | 용도 | 내용 |
|------|------|------|
| **왼쪽** | 제어 (설정/전송) | ON TIME 설정, 배수 설정, 순차 페이지, 트리거 Skip |
| **오른쪽 상단** | 장비값 — 조회 가능 | ON TIME (페이지별), 배수 (채널별), 현재 페이지 |
| **오른쪽 하단** | 장비값 — 조회 불가 | 설정 전송 시 반영되는 값 (Section, HOLD, Skip 등) |
| **하단** | 통신 로그 + 상태바 | 전체 너비 사용 |

## 2. 조회 가능 / 불가능 분류

### 조회 가능 (장비에서 읽어올 수 있음)
| 항목 | 명령 |
|------|------|
| 16CH ON TIME (페이지별) | `:1RA` / `:00R` |
| 16CH 배수 | `:1RU` |
| 현재 페이지 | `:00G` |

### 조회 불가 (설정만 가능, 전송 시 값 반영)
| 항목 | 설정 명령 |
|------|----------|
| Max Page | `:1P[값]` |
| 시작 페이지 | `:00P[값]` |
| HOLD ON/OFF | `:00H[0/1]` |
| Section ON/OFF | `:00CO[0/1]` |
| Section 시작/끝 | `:00CPS[값]` / `:00CPE[값]` |
| 트리거 Skip ON/OFF | `:00IO[0/1]` |
| Skip Time | `:00IT[값]` |
